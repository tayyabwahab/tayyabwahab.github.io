RAG-Powered SiteAssistant

The project was developed with Retrieval-Augmented Generation (RAG) technology using LangChain and the OpenAI API to help construction site workers quickly find answers to component-related queries from their respective manuals, eliminating the need to manually browse lengthy technical documents.

RAG Machine Learning OpenAI LLM Python Langchain Pinecone AWS

Project Overview

This project implements a RAG-based information retrieval application designed to support construction site workers in identifying and understanding various building components, including those that are new or unfamiliar. The system processes manuals for all relevant components, breaking them into smaller, searchable chunks and storing them as embeddings in a vector database for efficient semantic search. When a worker submits a query, the system retrieves the most relevant manual sections using similarity search and passes them to a language model via LangChain integrated with the OpenAI API. The language model then generates a clear, concise, and contextually accurate answer.

Key Features

  • Quick Reponse to Queries
  • Document Ingestion & Preprocessing
  • Vector Database
  • Effective Response using LLM

The system integrates multiple sensors and algorithms for robust target detection and tracking in dynamic environments.

Robotics Computer Vision Python Pytorch OpenCV AirSim YOLO DeepSORT

Project Overview

Designed with security and situational awareness in mind, the system empowers drones to detect, recognize, and track individuals in real time—particularly in crowded or high-risk areas—by leveraging facial recognition and full-body feature extraction capabilities. Single Image DepthNet

Single Image DepthNet is a modified U-Net architecture that takes a single RGB image as input and predicts its corresponding depth map, enabling scene understanding from 2D images without stereo input.

Computer Vision Machine Learning U-Net Python Pytorch OpenCV

Project Overview

Single Image DepthNet is a deep learning-based project focused on predicting depth information from a single RGB image using a modified U-Net architecture. The project is built with PyTorch and the public dataset NYU-DepthV2 was used to train and test the models efficiency. The U-Net model architecture consists of an encoder that captures contextual features from the input image and a decoder that reconstructs the spatial resolution to generate a dense depth map. The training pipeline includes image preprocessing, loss computation and model optimization using an Adam optimizer. During inference, the model predicts the relative depth of each pixel, producing a grayscale depth map that can be visualized to assess depth quality. This project demonstrates the application of convolutional neural networks (CNNs) for pixel-wise regression tasks and is useful for applications like 3D scene reconstruction, AR/VR, and autonomous navigation using monocular vision.

Key Features

  • Monocular Depth Estimation
  • Lightweight & Easy to Extend
  • Applicable in autonomous navigation, augmented reality, 3D reconstruction, and robotics using monocular vision systems.

Baccarat Game

This project is a computer vision–based and deep learning based intelligent system for real-time detection and tracking of playing cards in Baccarat games. Built for the Jetson Nano edge device, it utilises the famouse object detection model YOLOv5 integrated with custom tracking algorithm to classify card positions as BANKER or PLAYER, assign unique IDs, and maintain consistent tracking throughout gameplay.

Computer Vision Deep Learning Python Pytorch TensorRt Object Detection Jetson Nano

Project Overview

This project implements a specialized intelligent card detection and tracking system for monitoring Baccarat tables in real time. Designed for deployment on the NVIDIA Jetson Nano edge device, the system combines YOLOv5, an object detection model famous for its high speed and accuracy, integrated with a custom tracking algorithm to follow individual cards across video frames. The detection module identifies playing cards from a live camera feed, while the tracking module assigns each card a unique ID to maintain continuity throughout the game. The system further classifies cards into BANKER or PLAYER zones based on their location on the table, enabling role-specific analysis. Core components include the YOLOv5s model trained on a custom dataset of playing cards, the tracker for low-latency object persistence, a classification logic module for positional labeling, and an optimized video pipeline to run efficiently on Jetson Nano hardware.

Key Features

  • Real-Time Card Detection
  • Custom Tracking Algorithm
  • BANKER/PLAYER Classification
  • Unique ID Assignment
  • Lightweight & Efficient Pipeline
  • Optimized for Edge Deployment

Office Robo

Office Robo is a comprehensive ROS (Robot Operating System) based autonomous robot navigation project that implements advanced pathfinding and obstacle avoidance algorithms. The project features a simulated robot that can navigate through complex environments using a combination of go-to-point navigation and wall-following behaviors.

Robotics Artificial Intelligence ROS Python

Project Overview

Office-Robo is a sophisticated autonomous navigation system built on the Robot Operating System (ROS) framework, implementing advanced pathfinding and obstacle avoidance algorithms for indoor environments. It features a differential drive robot with a high-precision laser scanner for comprehensive environment perception, capable of navigating to target coordinates while intelligently avoiding obstacles through a hybrid approach that combines direct path navigation and wall-following behaviors. The system uses a state machine architecture with three main algorithms: a go-to-point navigation method, a wall-following routine that manages multiple obstacle scenarios, and a service orchestrator that dynamically switches between modes based on real-time sensor data. It supports modular communication via ROS services, includes full Gazebo simulation and RViz visualization, and offers mapping capabilities through gmapping SLAM.

Key Features

  • ROS-Based Architecture
  • Autonomous Indoor Navigation
  • Hybrid Navigation
  • Obstacle Avoidance
  • Mapping & Localization
  • URDF Robot Model

SecureFaceChain: Encrypted Facial Biometrics on Blockchain

SecureFaceChain is a blockchain-powered facial recognition authentication system that combines advanced machine learning with decentralized security. It uses a blockchain framework to store and validate encrypted facial feature data, with a Generative Adversarial Network (GAN) ensuring biometric privacy.

Computer Vision GAN Machine Learning Block Chain Face Recognition Python Mediapipe Pytorch OpenCV JavaScript Solidity

Project Overview

SecureFaceChain is a blockchain-based facial recognition authentication system that combines decentralized architecture with advanced machine learning techniques to protect biometric data and ensure secure identity verification. The system utilizes a blockchain framework to store and validate facial feature data in a tamper-proof and transparent manner, eliminating the risks associated with centralized databases. A Generative Adversarial Network (GAN) is integrated to encrypt and decrypt facial features, preserving the privacy and integrity of sensitive biometric information throughout the authentication process. By fusing the cryptographic strength of blockchain with the privacy-enhancing capabilities of GANs, SecureFaceChain delivers a robust, privacy-first solution for secure, decentralized, and trustworthy identity authentication in high-stakes or privacy-sensitive applications.

  • Decentralized Facial Data Storage

Watkins

Watkins is a cross-platform step counter and walking guidance application was deve for both Android and iOS platforms. The app leverages device sensors to detect steps and provides interactive walking instructions with audio-visual feedback.

Machine Learning Kotlin Swift C++

Project Overview

Watkins is an advanced step-counting and walking guidance application developed for both Android and iOS platforms. It utilizes mobile device sensors combined with a machine learning algorithm to intelligently estimate a user's steps based on real-time physical movement. The app not only tracks activity with precision but also enriches the user experience through interactive and engaging guidance. Upon launch, Watkins prompts users to enter the distance they wish to travel and their gender, allowing the system to personalize the walking session. Throughout the walk, the app provides real-time, step-by-step audio and visual instructions, supported by animated GIFs demonstrating walking, sitting, and turning actions for enhanced visual interaction. Once the walk is completed, Watkins delivers a detailed summary of the session, offering users insights into their trip and progress. This thoughtful combination of data-driven accuracy and user-friendly interactivity makes Watkins ideal for those seeking motivation, personalized fitness routines, or guided walking experiences.

Key Features

  • Machine Learning-Powered Estimation for Accurate Step Counting
  • Real-Time Walking Guidance
  • Custom Walk Configuration
  • Interactive Visual and Audio Feedback
  • Trip Summary Report
  • Cross-Platform Availability
